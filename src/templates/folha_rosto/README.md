# Templates de Folha de Rosto

Esta pasta deve conter os seguintes arquivos:

## Arquivos Necessários

1. **folha_rosto_edge.docx**
   - Template padrão para instituição EDGE
   - Usado para rubricas normais

2. **folha_rosto_vertex.docx**
   - Template padrão para instituição VERTEX
   - Usado para rubricas normais

3. **custos_incorridos_edge.docx**
   - Template para rubricas de "Custos Incorridos" (EDGE)
   - Usado quando a rubrica contém "custo" e "incorr"

4. **custos_incorridos_vertex.docx**
   - Template para rubricas de "Custos Incorridos" (VERTEX)
   - Usado quando a rubrica contém "custo" e "incorr"

## Variáveis do Template

Cada template deve conter as seguintes variáveis:

- `{instituicao}` - EDGE ou VERTEX
- `{projeto_codigo}` - Código do projeto
- `{projeto_nome}` - Nome do projeto
- `{pc_numero}` - Número da prestação de contas
- `{natureza_disp}` - Tipo de rubrica/natureza do dispêndio
- `{rubrica}` - Rubrica (compatibilidade)
- `{favorecido}` - Nome do favorecido
- `{cnpj}` - CNPJ/CPF do favorecido
- `{n_extrato}` - Número do extrato bancário
- `{nf_recibo}` - Número da NF ou recibo
- `{data_emissao}` - Data de emissão do documento
- `{data_pagamento}` - Data do pagamento
- `{valor_pago}` - Valor total pago
